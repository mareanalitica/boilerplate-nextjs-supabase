# STACK.md

## Visão Geral da Pilha Tecnológica

Este documento descreve a pilha tecnológica utilizada no projeto, abrangendo as principais linguagens, frameworks, bibliotecas e ferramentas.

### Linguagens de Programação
- **Frontend:** TypeScript, JavaScript
- **Backend:** TypeScript, JavaScript (Node.js)

### Frameworks e Bibliotecas
- **Frontend:** React, Next.js
- **Backend:** Express.js
- **UI/UX:** Tailwind CSS, Shadcn UI
- **State Management:** Zustand

### Backend as a Service (BaaS) & Banco de Dados
- **BaaS:** Supabase (Autenticação, Banco de Dados, Armazenamento, Funções Edge)
- **Banco de Dados:** PostgreSQL (gerenciado pelo Supabase)
- **ORM:** Prisma (para interação com PostgreSQL)

### Ferramentas de Desenvolvimento
- **Controle de Versão:** Git
- **Gerenciador de Pacotes:** npm, pnpm
- **Build Tools:** Webpack, Turborepo
- **Testes:** Jest, React Testing Library, Playwright
- **Linter/Formatter:** ESLint, Prettier

### Infraestrutura
- **Cloud Provider:** Vercel (Frontend), Supabase (BaaS, Database, Auth, Storage)
- **Containerização:** Docker (para ambientes de desenvolvimento e produção)

### CI/CD
- GitHub Actions
